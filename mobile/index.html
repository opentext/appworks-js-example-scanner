AppWorks Document Scanner Demo

You have three buttons below, all of which invoke AWScanner.scanDocument, but each receive results in a different way.

"File Path" will return a file path of the PDF document

"Base64" will return a base64 data representation of the PDF document

"Doc Provider" will invoke the document provider to send the PDF document to another app on the device

Result will appear here.
